Introduction¶

Kenning provides an API for deploying deep learning applications on edge devices using various frameworks for model training and compilation.

This documentation contains the following chapters:

  • Kenning section provides project description, installation steps and quick start guide,

  • Deep Learning deployment stack section describes the typical flow of deploying models on edge devices,

  • Using Kenning from command-line section describes the executable scripts available in Kenning,

  • ONNX support in deep learning frameworks section shows the support for ONNX model format in deep learning frameworks,

  • Sample autogenerated report section shows sample report generated using Kenning,

  • Kenning API section describes the Kenning API.
