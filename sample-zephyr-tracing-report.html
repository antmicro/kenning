    Sample Zephyr Tracing Report¶

    This section contains a sample performance report with Zephyr tracing section, generated by Kenning using Zephelin.

    The CI is set up as follows:

    • Environment: Github Actions

    • Task: Anomaly Detection on CATS dataset

    • Tracing tool: Zephelin

    • Compiler framework: TFLite

    • Platform: stm32f746g_disco board

    ¶

    Commands used¶

    Note

    This section was generated using:

    kenning test report \
    --cfg \
        /tmp/scenario-vae-tflite.yml \
    --measurements \
        results.json \
    --report-types \
        zephyr_traces \
    --report-path \
        /home/runner/work/kenning/kenning/docs/source/generated/sample-zephyr-tracing.md \
    --report-name \
    --to-html
